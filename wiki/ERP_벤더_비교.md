# ERP 벤더 비교 — 더존 vs 영림원

[[더존_아마란스10]] 와 [[영림원_K-System_Ace]] 는 국내 ERP 시장의 양대 옵션이며 양극(통합형 vs 연동형) 구도를 그대로 구현. 단순히 "어느 게 좋다"가 아니라 **회사가 어느 축을 선호하느냐** 를 먼저 결정해야 비교가 의미 있음 → [[통합형_vs_연동형]].

## 항목별 비교

| 항목 | [[더존_아마란스10]] | [[영림원_K-System_Ace]] |
|------|-------------------|------------------------|
| **출신 DNA** | 회계 시스템 → ERP 확장 | 한국형 ERP 전문 (국내 최초) |
| **추천 규모** | 스타트업~중견 (50~500명대) | 대기업·중견 |
| **그룹웨어/메신저** | **기본 통합** (한 플랫폼) | 별도 K-System GW 모듈 |
| **AI 기능** | ONE AI 적극 통합 | 명시적 AI 통합 부각 적음 |
| **클라우드** | SaaS 적극 | 별도 라인 SystemEver(SaaS) 또는 K-System Ace 구축 |
| **외부 연동** | 자체 생태계 중심 | **외부 솔루션 연동 폭 큼** (강점) |
| **건설/PMS** | KISS 일반 PM. 건축 특화 미명시 | **프로젝트 모듈 — 호기별 원가, 수불, 예산/실적** |
| **UI/사용성** | 웹 기반 모던 | 전통적, 기능 풍부 |
| **가격(외부 추정)** | 클라우드 월 ~100만원, 구축형 수천만~수억 | 비공개 (견적 기반) |
| **유지보수** | 월 자동패치, 분기별 업데이트 | 컨설팅 기반(상주/방문 선택) |
| **고객사 수** | 더존 그룹 전체 십수만 (아마란스10은 신제품) | 3,000여 개 |
| **보안** | AES-256, CSAP, 육군본부 | 명시 미상 |

## 시나리오별 우위

### 영림원 우위
- **프로젝트 단위 손익/원가** 가 핵심 KPI인 경우 ([[정보_일원화_패턴]] 의 손익 일원화 도메인)
- BIM/CAD/도서관리 등 **외부 시스템 연동** 다수
- 산업별 맞춤 컨설팅으로 깊은 커스터마이즈 필요
- 중견~대, 장기 ERP 자산화 관점

### 더존 우위
- 그룹웨어·메신저·결재·문서·회계를 **한 화면/한 로그인** 으로 묶고 싶은 경우 (이게 가장 강한 차별점)
- 회계·세무 신고 자동화 우선
- 클라우드 SaaS 선호, 빠른 도입
- AI(회의록·결재 자동검증·메일 분석) 활용 비중 높임 의도
- 재택/모바일/디지털 전환 추진 중

## SharedIT 등 IT 실무자 코멘트 요지
- 더존: 회계 강점. **비즈니스 룰이 더존 식으로 강제** 되는 경향
- 영림원: 컨설팅 의존도 큼. **PM·구축 파트너 역량이 결과를 좌우**
- 그룹웨어 따로 / ERP 따로 운영 시 **데이터 중복 + SSO 복잡도** 가 큼 → 통합형 가치 ([[정보_일원화_패턴]])
- 도입 후 1~2년 내 **안정화 기간 필수** → RFP 에 운영 책임 분담 명시

## 1차 분기점 의사결정 (전산실 총괄 관점)

```
[ ERP 도입 결정 ]

     ├─ 통합형 선호?  ──→ 더존 아마란스10 → 추가 검토: 건설 PMS 보완 방안

```
