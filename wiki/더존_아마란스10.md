# 더존 아마란스10 (Amaranth 10)

더존비즈온의 통합 비즈니스 플랫폼. **ERP + 그룹웨어 + 문서관리 + 프로젝트관리(KISS) + 생성형 AI(ONE AI)** 를 단일 플랫폼·단일 로그인으로 묶음. 회계 DNA 출신(더존)이지만 그룹웨어·메신저까지 **기본 통합** 한 것이 [[영림원_K-System_Ace]] 와의 핵심 차별점이며, [[통합형_vs_연동형]] 구도에서 통합형의 표상.

## 구성 모듈
- **ERP**: 회계(자동전표, 결산, 자금, 세무신고 자동화), 인사·급여(근태/연차/근로기준법 자동검증)
- **그룹웨어**: 메인 포털, 통합검색, **업무용 메신저·대화방**, 전자결재, 메일, 일정·자원, 화상회의·화면공유, 팩스/SMS
- **ONEFFICE** — 설치 없는 웹 오피스, 실시간 공동 편집
- **ONECHAMBER** — 클라우드 스토리지 기반 문서 라이프사이클(작성→보관→유통→폐기)
- **KISS** — 프로젝트 관리, 통합 대시보드 ※ 건설/건축 특화는 부재 → [[건설_PMS]] 보강 필요
- **ONE AI** — 자연어 통합검색, 요약·번역·이미지 생성, **회의록 자동작성**, 메일/대화방 분석, **결재 자동검증**, ERP 데이터 분석, ONE AI 템플릿(Low-Code)

## 도입 대상 (더존 공식 분류)
| 구분 | 인원 / 매출 |
|------|------------|
| 중소기업 | 1~120명 / 60~100억 |
| 중소·중견 | 70~200명 / 100~200억 |
| 중견·대기업·글로벌 | 200명 이상 / 200억 이상 |

추천 업종: 대기업, 병원·제약, 미디어·출판, 금융, 에너지, 정보통신, IT서비스, 제조, 공공/공기업, 재단·협회.
> ⚠ **건축·설계 사무소** 는 공식 추천 업종에 없음. PMS 영역은 KISS 모듈로 일반 PM만 커버. 건설업 원가관리 깊이는 [[영림원_K-System_Ace]] 의 프로젝트 모듈이 더 강하다는 평가.

## 가격 (외부 추정, 2025)
- **클라우드 SaaS**: 월 약 ₩1,000,000~ (인원/모듈) + 초기 컨설팅 수백만원
- **구축형**: 라이선스 + 구축비 별도, 연간 유지보수 라이선스의 15~20%
- 정부 지원사업 활용 시 최대 50~85% 보조 가능

## 강점
- **단일 로그인 통합** — ERP↔그룹웨어 데이터 중복 입력 제거 ([[정보_일원화_패턴]] 의 한 구현)
- AI 통합 (ONE AI) — 반복 업무 자동화
- AES-256, CSAP 인증, 육군본부 도입 → 보안 신뢰
- 매월 자동 패치 / 분기별 기능 개선
- 전국 파트너사 네트워크

## 약점
- **비용 부담** — 10인 미만 부적합
- **초기 안정성** — 출시 초 "오류 잦다"는 평가, 현재는 개선 중이나 완벽까지는 아직
- **학습 곡선** — 모듈 범위 넓음, 회계·인사 모듈별 교육 필요
- **글로벌 호환성 약함** — SAP/Oracle 호환 필요 시 부적합
- **건축설계 특화 모듈 부재** — KISS 의 일반 PM 만 제공

## 도입 사례
- 이노그리드 (내부회계 효율화), 부민병원 (모바일 출퇴근), GS스포츠 (외근 모바일), (주)타이타늄22 (재택)

## ONE AI 의 가치 — 의외의 시각
ONE AI 가 풀려는 문제(회의록 자동작성, 결재 자동검증, 데이터 분석)는 [[LLM_Wiki_개념]] 이 푸는 문제와 본질이 같다 — 조직의 흐트러진 텍스트를 한 곳에 정리하고 다음 의사결정 때 자동 참조되게 만드는 것. 차이는 ONE AI 가 **벤더 통합형** 으로 구현하고, [[LLM_Wiki_개념]] + Claude + MCP 가 **사용자 연동형** 으로 구현한다는 점뿐. → [[통합형_vs_연동형]]

## 의사결정 시 핵심 질문
- 그룹웨어·메신저·결재·문서·회계를 한 화면으로 묶는 가치가 락인 위험보다 큰가? ([[정보_일원화_패턴]])
- 건축설계 특화 PMS 부재를 어떻게 보완할 것인가? (KISS + 외부 PMS 연동? 또는 영림원으로?)
- ONE AI 의 회의록·결재·데이터 분석을 외부(Claude + MCP) 로 분리해도 충분한가?
