# 워처 Race 가드 테스트

kouwiki ingest 엔진이 동일한 raw 파일을 여러 경로로 동시에 감지하더라도, 단 한 번만 위키 페이지로 생성되도록 보장하는 메커니즘을 테스트하고 검증하는 절차입니다. 이는 [[건강검진]]에서 강조하는 시스템 일관성 유지의 핵심 요소입니다.

## 개요
본 테스트는 파일 시스템 워처(polling)와 수동 스캔 등 여러 입력 경로가 동시에 `raw/` 디렉토리 내의 동일한 파일을 감지했을 때, 시스템이 중복 없이 정확히 한 번만 해당 파일을 처리하는지 확인합니다. 이는 위키 데이터의 무결성과 일관성을 유지하는 데 필수적입니다.

## 검증 항목
- **단일 페이지 생성**: 동일한 raw 파일에 대해 `wiki/` 디렉토리에 단 하나의 위키 페이지 파일만 생성되어야 합니다.
- **index.md 단일 라인 추가**: `index.md` 파일의 해당 카테고리에 관련 페이지 링크가 단 한 줄만 추가되어야 합니다.
- **log.md 단일 행 추가**: `log.md` 파일에 해당 ingest 작업에 대한 기록이 단 한 행만 추가되어야 합니다.
