# 영림원 K-System Ace

영림원소프트랩의 한국형 ERP. **국내 최초 한국형 ERP** 30년 업력. "현장 밀착 프로세스 기반 자율형 ERP" 표방. 산업별 특화 모듈 + 외부 솔루션 연동 폭이 강점, 특히 **프로젝트/건설/원가관리** 가 차별점. [[통합형_vs_연동형]] 구도에서 연동형의 표상.

## 모듈 구성

### 기본 모듈
- 재무회계 — 결산, 마감 자동화
- 인사급여 — 급여, 근태
- 영업물류 — 판매, 재고, 배송
- 생산구매 — 생산 계획, 구매
- 원가관리 — 원가 추적·분석
- **프로젝트** — 호기별 매출원가, 프로젝트별 수불, 예산/실적, 견적, A/S, 계측기 이동 → [[건설_PMS]] 적용 시 핵심

### 확장 솔루션
- K-System BI (경영정보 시각화), K-System Mobile, K-System MES, **K-System GW (그룹웨어)**, K-Studio (개발 도구)
> ⚠ 그룹웨어/메신저는 **별도 K-System GW 모듈**. [[더존_아마란스10]] 처럼 기본 통합되지 않음 → 별도 도입·연동 필요. 이 차이가 [[통합형_vs_연동형]] 분기점.

## 산업별 특화 (강점)
| 산업 | 특화 기능 |
|------|----------|
| **건설/PMS** | 호기별 매출원가, 프로젝트별 수불, 예산/실적, 견적, A/S, 계측기 이동 → [[건설_PMS]] |
| 제약 | 일련번호, 거래처 원장, Audit Trail |
| 화학 | 안전/보건, 폐기물, 화학물질 |
| 식품/유통 | 유통기한 추적, 의제매입 공제 |
| 공공 | 예산, 지출결의, 국고보조금 |
| 자동차부품 | 자동 생산계획, 매출/매입 이월 |

## 외부 솔루션 연동 (핵심 강점)
비즈플레이, NICE신용정보, 사방넷, 스파로스클라우드EDI, 솔코PDM, 전자세금계산서, 카카오 알림톡, FTA-PASS, 세일즈포스CRM, 네이버클로바OCR 등.

→ **"다양한 외부 솔루션과의 유연한 연동성"** 이 영림원의 핵심 셀링 포인트. 이는 [[통합형_vs_연동형]] 의 연동형 가치를 그대로 구현.

## 클라우드 옵션 — SystemEver
영림원의 SaaS ERP. K-System Ace 와 별도 라인. 중견 이하 / SaaS 선호 기업 대상.

## 도입 대상
- 국내외 **3,000여 개 고객사**
- 공식 추천: **대기업·중견기업** (인원·매출 기준 명시 없음)
- 산업별 맞춤형 모듈 활용 → 기업 고유 프로세스에 맞춤 구성
- 공공/비영리/중소 전용 버전 별도
- **방문형 구축** 가능 (중소 친화)

## 차별점
1. **프로세스 중심 설계** — 구축→업무→마감→경영개선 시각화
2. **경영분석 통합** — ERP 데이터 기반 6개 모델 (생산/손익/경영/구매/영업/자금)
3. **30년 업력** — 국내 최초 한국형 ERP
4. **TO-BE 프로세스 메뉴화** — 프로세스 자체를 시스템 자산화
5. 상주형/방문형 컨설팅 선택

## 강점·약점

| 강점 | 약점 |
|------|------|
| 산업별 특화 모듈 깊이 (특히 [[건설_PMS]]) | 그룹웨어/메신저 기본 미통합 |
| 외부 솔루션 연동 폭 | 클라우드 vs 온프레미스 구도 페이지에 명확치 않음 |
| 프로젝트/원가관리 | 가격·도입 기간 비공개 |
| 30년 안정성 | UI 가 다소 전통적 |

## 건축설계사무소 관점 — 우위 시나리오
- **프로젝트 단위 손익/원가** 가 핵심 KPI ([[정보_일원화_패턴]] 의 손익 일원화 적용 도메인)
- BIM/CAD/도서관리 도구 등 **외부 시스템 연동** 다수 필요
- 산업별 맞춤 컨설팅 통한 **프로세스 깊은 커스터마이즈** 가 필요한 경우
- 회사 규모 큰 편 (중견~대), 장기 ERP 자산화 관점

## RFP 시 확인 필수 항목
- 그룹웨어·메신저 통합 시 **K-System GW** 도입 비용 별도 산정
- 외부 시스템 연동 가능 목록 — BIM/CAD 도구 연결 사례 요청
- 프로젝트 모듈의 **공수(MM) 기반 원가** 적용 가능 여부 ([[건설_PMS]] 참조)
