# 사내 인쇄·복합기 사용 가이드

해마건축 본사 5층 복합기 사용법을 정리한 가이드입니다. 인쇄 및 스캔 방법을 다룹니다.

## 인쇄 방법
1. 사용자 ID 입력
2. 비밀번호 입력
3. 출력 큐에서 문서 선택

## 스캔 방법
1. 원고 올리기
2. 자기 이메일 자동 발송 또는 USB 저장
