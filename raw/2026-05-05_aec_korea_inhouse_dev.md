## 한 줄 요약
국내 건축설계사무소 중 **정림건축**이 BIM·컴퓨테이셔널 디자인·디지털 트윈 영역에서 가장 가시적인 사내 R&D + 교육 프로그램(JAdU)을 운영. **간삼건축**도 BIM 적용 실적 1위. 다만 사내 개발 코드/도구를 외부에 공개하는 사례는 매우 적음 → **블로그·교육 프로그램 동향 추적**이 주된 정보원.

## 1. 정림건축 — 가장 적극적인 디지털 전환 사례

### 1-A. Junglim Innovation (혁신본부)
**공식 블로그**: <https://innovation.junglim.com>

#### 사례: BIM 규모검토 Package
- **문제**: 초기 설계 단계에서 CAD/SketchUp/Rhino/Excel/PowerPoint 다중 도구 사용 → 면적 변경 시 수치 불일치, 반복 작업
- **해법**: Revit 2017 기반 자체 개발 패키지, **단일 정보원(Single Source of Truth)** 원칙 — 본 위키 [[정보_일원화_패턴]] 의 BIM 도메인 구현
- **기능**:
  - 매스↔평면↔면적 연동 (매스 수정 시 면적 자동 반영)
  - 태그↔용도 연동
  - Material 기반 컬러링
  - Excel 연계 (초기 설정 후 데이터만 갱신)
  - **BIM Law 데이터** 불러오기 (법규 자동 체크)
- **조직**: Junglim Innovation 본부 BIM팀 (BIM Manager 정혜수)
- **확산**: J-edu 교육 영상으로 업계 배포

### 1-B. JAdU — 정림 아카데미 디지털 유닛
**홈페이지**: <https://www.jacademyhub.com/>

- **목표**: 디지털 트윈, BIM, 컴퓨테이셔널 디자인 등 **하이테크 친환경 설계 인재 양성**
- **시작**: 2024년 5월 1기 → 2025년 3기 진행 → 2026년 4기 모집
- **국가 인증**: K 디지털 트레이닝 선도기업 아카데미 / 첨단산업·디지털 선도기업 아카데미
- **커리큘럼**: 600시간 집중 + Project Based Learning
  - 모듈러 디지털 트윈 프로젝트
  - ZEB 친환경 건축 시뮬레이션
  - BIM 교육 + 자격증
  - AI 특강 + D5 렌더링
  - 3D 프린팅 실습
- **취업 연계**: JAdU 수료생 → 정림건축 입사 사례 다수
- **JAH (정림아카데미허브)**: 교육 프로그램 기획·운영 조직

### 1-C. 정림 + 티랩스 협약
- 메타버스 서비스 개발 + 인프라 구축 사업 협력
- 디지털 트윈 영역 강화

### 1-D. 사용 기술 스택
- **Revit + Dynamo** (메인)
- Python (Dynamo 노드, Revit API)
- C# (Revit API)
- D5 렌더링 / AI 특강 도입

## 4. 한국 BIM·디지털 전환 정책

### 국토교통부 — BIM 기반 건설산업 디지털 전환 로드맵 (2021~)
- **국가 목표**: 2030년 전 사업 BIM 의무화
- 공공발주 BIM 의무 → 민간 확산
- BIM 기반 인허가 자동화 (한 달 → 15일 단축 목표)

### KICT 인공지능 기반 건축설계 자동화 기술개발
- 설계 생산성 60% 향상
- 건축설계 의사결정 45% 자동화
- 설계 검토 자동화 30%+

### 경희대 — 개방형 BIM 기반 자동화 + 첨단 유지관리 기반기술 개발
- 산학 협력 R&D
- 자동 법규 체크 핵심 과제

## 7. 시사점 — 300명 규모 건축설계사무소 IT 책임자 관점

### 정림 모델의 적용 가능성
정림건축의 **혁신본부 + JAdU** 구조는 300명 규모 사무소에 일부 차용 가능:
1. **소규모 디지털 디자인 그룹 1~3명** 신설 (정림 혁신본부 BIM팀 = 1인 시작)
2. **사내 교육 프로그램**으로 도구 사용 확산 (J-edu 모델)
3. 첫 사내 도구는 **반복 + 단일 정보원 위반 영역** 부터 ([[정보_일원화_패턴]] 의 가장 큰 비효율 지점)
4. 사내 도구 → 외부 공개(블로그)로 **채용 브랜딩** 효과 (정림이 검증)

### 글로벌 ([[2026-05-05_aec_global_inhouse_dev|Foster·Gensler]]) 와의 비교
| 항목 | Foster + Partners | 정림건축 | 본 사무소(300명) |
|------|------------------|---------|-----------------|
| R&D 팀 규모 | 다학제 수십명 | 혁신본부 (소규모) | 1~3명 권장 |
| 자체 도구 공개 | 비공개(논문/블로그만) | 일부 공개 | 가능(정림 모델) |
| 외부 의존 도구 | Rhino + 상용 BIM 만 | Revit + Dynamo 중심 | Revit/CAD + Speckle PoC 권장 |
| 교육 프로그램 | 사내 알파/베타 | **외부 인재 양성(JAdU)** | 사내 우선 |

### 추천 우선순위
1. **즉시 가능**: Junglim Innovation 블로그 RSS 구독, JAdU 4기 견학 신청
2. **3개월 내**: BIM 자동화 1순위 영역 식별 → Revit + Dynamo + Python 으로 PoC
3. **6개월 내**: 사내 디지털 디자인 그룹 1명 채용 또는 보직 전환
4. **12개월 내**: Speckle 등 외부 플랫폼 연동 검토 (벤더 락인 회피)
