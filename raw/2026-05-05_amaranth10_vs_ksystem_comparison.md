## 한 줄 요약
- **더존 아마란스10** — 회계 DNA 기반, **올인원 통합 플랫폼**, AI/클라우드 등 최신 기술. 그룹웨어/메신저 기본 통합.
- **영림원 K-System Ace** — 산업별 **맞춤형/특화 모듈** 깊이, 외부 연동 폭, 프로젝트/원가관리 강점. 그룹웨어 별도.

## 건축설계사무소 관점에서의 의사결정 포인트

### 영림원 우위가 예상되는 시나리오
- 프로젝트 단위 손익/원가가 핵심 KPI인 경우
- BIM/CAD/도서관리 도구 등 외부 시스템 연동이 다수 필요한 경우
- 산업별 맞춤 컨설팅을 통한 프로세스 깊은 커스터마이즈가 필요한 경우
- 회사 규모가 큰 편이고(중견~대) 장기 ERP 자산화 관점

### 더존 우위가 예상되는 시나리오
- 그룹웨어·메신저·결재·문서관리·회계를 **한 화면/한 로그인**으로 묶고 싶은 경우 → 가장 강력한 차별점
- 회계·세무 신고 자동화가 우선 가치
- 클라우드 우선, 빠른 도입(SaaS) 선호
- AI(회의록 자동, 결재 자동검증, 메일 분석 등) 활용 비중 높임 의도
- 재택/모바일/디지털 전환 추진 중

### 공통 검토 항목 (벤더 무관)
- 프로젝트 회계: **수주 → 계약 → 진행률 → 손익 → 정산** 전 과정 시스템 추적 가능 여부
- 도서/도면 보관(아카이빙) 및 버전 관리
- 외주(외주 건축사·구조·설비 등) 비용 처리 워크플로
- 인허가 일정 추적
- BIM/CAD 도구와의 인터페이스 (특히 문서 메타데이터 연결)
- 모바일에서 결재/근태/현장 업무

## SharedIT 등 IT 실무자 코멘트 요지
- "더존은 회계가 강하다. 다만 비즈니스 룰이 더존 식으로 강제되는 경향"
- "영림원은 컨설팅 의존도가 크다. 프로젝트 매니저·구축 파트너 역량이 결과를 좌우"
- "그룹웨어 따로/ERP 따로 운영하면 데이터 중복·SSO 복잡도가 커짐 → 통합형 도입의 가치"
- "도입 후 1~2년 내 안정화 기간 필수. RFP에 운영 책임 분담을 명확히 명시"

## RFP에 반드시 들어가야 할 항목 (체크리스트)
1. 프로젝트(현장) 단위 매출/원가 마감 자동화
2. 결재 모바일 지원 + 외근/재택에서 작동
3. 외주 비용 → 계약 → 세금계산서 → 지급 까지 자동 연결
4. SSO / Active Directory 연동
5. 메신저/메일 / 문서관리 통합 또는 표준 API 제공
6. 데이터 외부 반출(Export) 정책 및 형식
7. 마이그레이션 시 기존 회계/PMS 데이터 이관 범위와 책임
8. 패치/업그레이드 주기 및 다운타임 정책
9. 보안 인증 (CSAP, ISO27001, ISMS 등)
10. SLA 및 장애 보상 조건
