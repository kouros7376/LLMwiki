## 한 줄 요약
더존비즈온의 통합 비즈니스 플랫폼. **ERP + 그룹웨어 + 문서관리 + 프로젝트관리 + 생성형 AI** 를 하나의 클라우드 플랫폼에 묶음. "Amaranth"(시들지 않는 꽃) + "10"(완성) 의미.

## 구성 (시스템 아키텍처)
한 플랫폼 안에 다음 기능 묶음:
- **임직원 업무관리** — 메인 포털, 통합검색, 일정·자원 예약
- **ONEFFICE** — 설치 없는 웹 오피스 (실시간 공동 편집, 클라우드 문서)
- **ONECHAMBER** — 클라우드 스토리지 기반 문서 라이프사이클(작성→보관→유통→폐기) 중앙 관리
- **KISS (Knowledge Information Sharing System)** — 프로젝트 관리, 통합 대시보드
- **결재 멀티뷰** — 전자결재 워크플로우
- **ONE-BOOK** — 다중 문서 동시 관리
- **ONE AI** — 생성형 AI (검색·요약·번역·이미지·회의록 자동 생성·결재 자동검증·ERP 데이터 분석)

## 주요 모듈

### ERP 영역
- 회계: 자동전표처리, 전표/장부, 결산/재무제표, 고정자산, 자금관리, 세무신고 자동화
- 인사·급여: 인사/근태/연차/급여/세무, 근로기준법 자동 검증
- 경비 청구·자금 관리, 손익계산서·경비 분석

### 그룹웨어
- 메인 포털, 통합검색
- **업무용 메신저, 대화방**
- 전자결재, 전자메일, 게시판
- 화상회의, 화면공유
- 일정관리, 자원 예약
- 팩스/SMS

### 문서관리
- ONEFFICE: 웹 기반 문서 작성 (설치 불필요)
- ONECHAMBER: 클라우드 중앙 보관, 권한·링크 공유
- 라이프사이클 관리

### AI (ONE AI)
- 자연어 통합 검색
- 문서 작성·번역·요약·이미지 생성
- 메일·대화방 분석/작성 지원
- 화상회의 텍스트 변환·회의록 자동 작성
- 전자결재 문서 자동 검증
- ERP 데이터 분석·비교
- ONE AI 템플릿 (Low-Code 맞춤형)

## 가격 (2025년 기준, 외부 리뷰)
**클라우드 SaaS형**
- 월 이용료: 약 ₩1,000,000 ~ (인원/모듈 규모별)
- 초기 컨설팅비: 수백만원대
- 서버/유지보수 포함, 자동 업그레이드

**구축형 On-Premise**
- 라이선스 일시금 + 구축비(견적 별도)
- 연간 유지보수료: 라이선스의 15~20%
- 정부 지원사업 활용 시 최대 50~85% 보조 가능

## 장점
- 회계·인사·결재·메신저까지 단일 로그인 통합 → 데이터 중복 입력 제거
- 웹/모바일 동등 지원, 재택근무 친화
- AI 통합으로 반복 업무 자동화
- AES-256 암호화, CSAP 인증, 육군본부 도입 → 보안 신뢰도
- 매월 자동 패치, 분기별 기능 개선
- 전국 파트너사 네트워크 + 1670-0999 고객센터

## 단점·페인포인트 (외부 리뷰)
- **비용 부담**: 클라우드 월 100만원대~, 10인 미만 영세기업 부적합
- **초기 안정성**: 출시 초기 "오류가 끊임없이 나온다" 평가, 현재는 개선됐으나 완벽까지는 아직
- **학습 곡선**: 모듈 범위 넓음 → 회계/인사 별 교육 필요, 용어·프로세스 적응 시간 소요
- **글로벌 호환성**: SAP/Oracle 호환 필요한 글로벌 기업에는 부적합
