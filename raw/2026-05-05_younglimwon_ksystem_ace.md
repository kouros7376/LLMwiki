## 한 줄 요약
영림원소프트랩의 한국형 ERP. **국내 최초 한국형 ERP** 30년 업력. "현장 밀착 프로세스 기반 자율형 ERP" 표방. 산업별 특화 모듈이 강점, 특히 **프로젝트/건설/원가관리**가 차별점.

## 모듈 구성

### 기본 모듈
- **재무회계** — 결산 관리, 마감 자동화
- **인사급여** — 급여, 근태
- **영업물류** — 판매, 재고, 배송
- **생산구매** — 생산 계획, 구매
- **원가관리** — 원가 추적·분석
- **프로젝트** — 호기별 매출원가, 프로젝트별 수불, 예산/실적, 견적, A/S, 계측기 이동

### 확장 솔루션
- **K-System BI** — 경영정보 시각화
- **K-System Mobile** — 모바일 업무
- **K-System MES** — 생산 실행
- **K-System GW** — 그룹웨어 (별도 모듈)
- **K-Studio** — 개발 도구

> ⚠️ **그룹웨어/메신저**: 별도 K-System GW 모듈로 제공. 아마란스10처럼 기본 통합되지는 않음 — 별도 도입·연동 필요.

## 산업별 특화

### 건설/PMS 관련
공식 페이지 "프로젝트" 섹션:
- 호기별 매출원가관리
- 프로젝트별 수불관리
- 예산/실적 관리
- 견적관리, A/S 관리
- 계측기 이동관리

→ 건설업 원가·현장·하도급 관리에 강점이 있다는 평가 다수.
다만 건축**설계** 사무소 특화 모듈은 별도 명시 없음 → 컨설팅을 통한 커스터마이즈 필요.

### 기타 산업 특화
| 산업 | 특화 기능 |
|------|----------|
| 제약 | 일련번호, 거래처 원장, Audit Trail |
| 화학 | 안전/보건, 폐기물, 화학물질 |
| 식품/유통 | 유통기한 추적, 의제매입 공제 |
| 공공 | 예산, 지출결의, 국고보조금 |
| 자동차부품 | 자동 생산계획, 매출/매입 이월 |

## 외부 솔루션 연동 (강점)
비즈플레이, NICE신용정보, 사방넷, 스파로스클라우드EDI, 솔코PDM, 전자세금계산서, 카카오 알림톡, FTA-PASS, 세일즈포스CRM, 네이버클로바OCR 등

→ "다양한 외부 솔루션과의 유연한 연동성"이 영림원의 핵심 강점으로 강조됨.

## 도입 대상
- 국내외 **3,000여 개 고객사** (영림원 자체 발표 2,600개~3,000개)
- 공식 추천: 대기업·중견기업
- 산업별 맞춤형 모듈을 활용해 기업 고유 프로세스에 맞게 구성 가능
- 공공/비영리/중소기업 전용 버전 별도 제공
- **방문형 구축**도 가능 (중소 친화적)

## 클라우드 옵션 — SystemEver
영림원의 SaaS ERP. K-System Ace 와는 별도 라인. 중견 이하 / SaaS 선호 기업 대상.

## 차별점
1. **프로세스 중심 설계** — 구축 → 업무 → 마감 → 경영개선 프로세스 시각화
2. **경영분석 통합** — ERP 데이터 기반 6개 모델 (생산/손익/경영/구매/영업/자금)
3. **30년 업력** — 국내 최초 한국형 ERP
4. **TO-BE 프로세스 메뉴화** — 프로세스 자체를 시스템 자산화
5. **상주형/방문형 컨설팅 선택**

## 강점·약점 요약
| 강점 | 약점 |
|------|------|
| 산업별 특화 모듈 깊이 | 그룹웨어/메신저 기본 미통합 (별도 모듈) |
| 외부 솔루션 연동 폭 | 클라우드/온프레미스 아키텍처 페이지에 명확히 안내 안 함 |
| 프로젝트/원가 관리 | 가격·도입 기간 비공개 |
| 30년 안정성 | UI는 더존 대비 다소 전통적 평가 |
| 대기업·중견 도입 사례 풍부 | |

## 도입 검토 포인트 (건축설계사무소 관점)
- 프로젝트 단위 손익관리·원가관리가 핵심이라면 영림원이 우위 평가 다수
- 다만 그룹웨어/메신저 통합은 별도 K-System GW 또는 외부 도구(예: Slack, Teams) 연동으로 풀어야 함
- 외부 시스템 연동 폭이 넓어 BIM/CAD/도서 관리 도구 연결 시 유리할 수 있음 → 사전 RFP 단계에서 연동 가능 여부 별도 확인 필수
